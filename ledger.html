Project Overview

Land Ledger Transaction is a comprehensive blockchain-based land registry system developed using Hyperledger Fabric. The system securely manages land ownership, mortgage approvals, tax processing, and ownership transfers while ensuring transparency and immutability of records.

The platform integrates with Bhutan's National Digital Identity (NDI) platform for secure user authentication and implements role-based access control for different stakeholders including landowners, government authorities, and financial institutions.

Technical Implementation

The system was built using a microservices architecture with Hyperledger Fabric as the underlying blockchain framework. Smart contracts (chaincode) handle business logic for parcel registration, sale initiation, and transaction processing.
